Tasks

Your assignment is to create, test and validate HTML5 (i.e. HTML) documents that can be used as your personal web site that you could use as an online CV. It will contain a home page and at least three (but no more than five) content pages.

  1. Propose a page layout design appropriate for your target audience. Include a storyboard/low fidelity prototype.
  2. Describe the organisation of files in your site; the overall layout of pages in your site and how they link to each other.
  3. Code and style your web pages using HTML and CSS using a text editor only. Do NOT use a WYSIWYG (What You See Is What You Get) HTML editors.

Required components of the (displayed) pages include:

  • One page displaying tables with tabular information (Tables must be visible and have borders.)
  • Display images and image maps
  • Display unordered, ordered list and description lists
  • One e-mail hyperlink
  • One external hyperlink
  • Association with external style sheet (.css file)

BOTH Mozilla Firefox and Internet Explorer must render each page.
